FAQ

Contact page questions

These short answers explain how the placeholder form is meant to be used.

Does the contact form send email already?

Not yet. The current version is a front-end placeholder so the site works fully as a static project. You can connect it later to Formspree, Netlify Forms, Basin, or another service.

Can I change the form fields?

Yes. You can add dropdowns, budget fields, or calculator request fields without changing the overall site structure.

Why include a contact page on a calculator site?

It helps with trust, brand signals, and ad-network readiness while giving visitors a clear place to send suggestions.

Where should I start if I only need a calculation?

Head back to the homepage or open the timesheet calculator directly.
